# 🚀 빠른 시작 가이드

## 📋 1단계: 프로젝트 설정

### 폴더로 이동
```bash
cd todo-nextjs-app
```

### 의존성 설치
```bash
npm install
```

## 🎯 2단계: 개발 서버 실행

```bash
npm run dev
```

브라우저에서 http://localhost:3000 열기

## ✅ 3단계: 기본 사용법

### 할 일 추가하기
1. 상단 입력창에 "Next.js 프로젝트 완성" 입력
2. 우선순위 선택 (예: "높음")
3. "추가" 버튼 클릭 또는 Enter

### 고급 옵션 사용하기
1. "고급 옵션" 클릭
2. 설명, 마감일, 카테고리, 태그 입력
3. "추가" 버튼 클릭

### 할 일 관리하기
- **완료 표시**: 체크박스 클릭
- **수정**: "수정" 버튼 → 내용 수정 → "저장"
- **삭제**: "삭제" 버튼 → 확인
- **복사**: "복사" 버튼

### 필터링 & 검색
- **필터**: "전체" / "진행중" / "완료됨" 버튼
- **정렬**: 드롭다운에서 선택
- **검색**: 검색창에 키워드 입력

### 데이터 백업
- **내보내기**: "내보내기" 버튼 → JSON 파일 다운로드
- **가져오기**: "가져오기" 버튼 → JSON 파일 선택

## 🎨 4단계: 커스터마이징 (선택)

### 색상 변경
`tailwind.config.js` 파일의 `colors` 섹션 수정

### 기본 우선순위 변경
`components/TodoInput.tsx`의 `useState('medium')` 부분 수정

### 저장소 변경
`lib/storage.ts`에서 LocalStorage 대신 다른 저장소 구현

## 📦 5단계: 프로덕션 빌드

```bash
npm run build
npm start
```

## 🎉 완료!

이제 강력한 할 일 관리 앱을 사용할 준비가 되었습니다!

## 💡 유용한 팁

- **키보드 단축키**: Enter 키로 빠른 추가
- **마감일**: 날짜 선택 시 지연/임박 알림 자동 표시
- **태그**: #태그명 형식으로 검색 가능
- **카테고리**: 한 번 입력하면 자동완성 지원
- **백업**: 주기적으로 내보내기 권장

## ❓ 문제 해결

### 포트 충돌
다른 포트 사용: `npm run dev -- -p 3001`

### 의존성 오류
캐시 삭제: `rm -rf node_modules package-lock.json && npm install`

### 빌드 오류
TypeScript 체크: `npm run build`
